# ArianeAI

Application Nuxt 4 avec support MCP (Model Context Protocol).

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

L'application sera accessible sur [http://localhost:3000](http://localhost:3000)

Le serveur MCP sera accessible sur [http://localhost:3000/__mcp/sse](http://localhost:3000/__mcp/sse) en mode développement.

## Build

```bash
npm run build
```

## Génération statique

```bash
npm run generate
```
